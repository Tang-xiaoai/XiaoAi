##面向对象三大特性
1.封装与隐藏（Python中无法完全限制访问）
隐藏一个对象的属性：1.修改对象的属性名（弱封装，象征意义上的封装。例如hidden_name，不能用name 访问，可以用hidden_name访问）2.修改属性名2，__name，不能通过对象.__name访问，但是可以通过 对象._类名__属性名访问
处理隐藏的属性：getter和setter方法，如：get_name
反正无法完全隐藏，一般用_属性名来示意该属性是私有属性
property装饰器作用：省去（）调用
2.继承，父类中的所有属性和方法都会被子类继承，包括特殊方法。
父类中的方法，子类中也有，构成重写。
在子类中，可以通过super()调用当前类的父类，且调用方法时不用传self参数
可多重继承，class C（A，B），如果父类中有同名方法，优先第一个。如果继承很多，类似于深度优先查找，且之前查找过的类不再查找。
3.多态

##异常
处理异常，try语句
try:
    try中放置的是可能出错的代码
except:
    except中放置的是出错以后的处理方式
    except NameError： 则会捕获NameError异常
    except 还可以继续写，不限数量
    Exception是所有异常类的父类，except Exception会捕获所有异常
else:
    没有出错继续执行的语句
finally:
    代码块，不管是否出错都会执行
异常的传播，由内向外
    程序运行出现异常以后，所有的异常信息会被保存在一个专门的异常对象中
      异常的传播就是异常对象的实例不断返回上一级的过程

    raise 用于向外部抛出异常，后可以跟一个异常类或者异常类的实例
