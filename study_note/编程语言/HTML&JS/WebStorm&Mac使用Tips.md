# 快捷键

## 编辑

| keyboard shortcut  | 作用                 |
| ------------------ | -------------------- |
| Command+Option+L   | 代码格式化(智能缩进) |
| Command+Shift+⬆️或⬇️ | 选中代码行上下移动   |
| Shift+回车         | 快速另起一行         |
| Command+D          | 复制当前行到下一行   |
| Command+Delete     | 删除当前行           |
