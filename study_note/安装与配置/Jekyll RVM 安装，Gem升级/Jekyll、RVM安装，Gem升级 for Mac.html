0.查看是否安装了gem
gem -v

gem update --systemgem更新

1.通过gem安装jekyll
gem install jekyll


Ruby版本有问题，需要升级Ruby。

2.Ruby升级

Ruby升级参考链接
1）安装RVM
curl -L get.rvm.io | bash -s stable

国外网站，所以速度比较慢，一次不行可以多试几次。
2）配置环境变量
source ~/.profile
rvm -v查询rvm版本信息

3）查询支持的ruby版本信息

4）指定版本安装
rvm install ruby-2.5.7

国外镜像下载太慢了，添加国内镜像
echo "ruby_url=https://cache.ruby-china.com/pub/ruby" > /Users/tangaihui/.rvm/user/db
gem source -l查看gem默认
下载镜像
gem sources --add https://gems.ruby-china.com/ --remove https://rubygems.org/
默认为：https://rubygems.org/ 记得改回去
5）尝试安装
rvm install ruby-2.5.7

5）查看安装版本信息
ruby -v

3.再次安装
jekyll
查看版本信息

4.检测
（1）在～目录下，创建博客相关文件，名如myblog
jekyll new myblog

ps:这里我多建了一个myblog文件夹，其实不用。
（2）进入jekyll创建的博客文件夹,jekyll serve开启jekyll服务，自动创建博客。
