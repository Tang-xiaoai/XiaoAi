1.使用apt安装（非root用户啊在命令前加sudo）
apt install nginx
2.查询nginx版本
nginx -v

3.启动关闭nginx命令
service nginx start 启动nginx
service nginx reload 重新加载nginx配置文件
nginx -s reopen 重启nginx
nginx -s stop 停止nginx
4.验证nginx是否配置成功
打开浏览器，输入localhost:80如果能跳转到nginx欢迎页面则成功
